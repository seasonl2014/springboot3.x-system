# 基于SpringBoot3.x+Vue3.x整合脚手架

## 1.1 脚手架功能说明

主要功能有：后台首页展示、个人信息展示和修改、用户管理、角色管理、日志管理等。

## 1.2 技术栈

1. web框架：SpringBoot3.X
2. 数据库框架：Sping Data JPA
3. 数据库：MySql8.0
4. 项目构建工具：Maven、vite
5. 前端框架：Vue3.X、element plus、pina、axios、Vue Router
6. 数据图表：ECharts

## 1.3 数据库表

本项目有3张表，分别如下：

1、用户表（sys_user）

2、角色表（sys_role）

3、日志表（sys_log）
